Расширяемый модуль безопастности GSM/GPS (на основе A9).
1. Основное описание
 - устройство предназначенное для обмена информацией и конфигурации через GSM;
 - в состав информации предназначенной для обмена входят данные GPS;
 - устройство является портативным;
 - весь обмен который осуществляется через GSM может осуществляться и через USB;
 - расширение функциональности модуля осуществляется за счёт специального протокола сценариев;
 - без расширения имеет базовую функцию маяка.

2. Базовая функция "Маяк".
 - после включения производится инициализация GSM модуля:
   - проверяется наличие SIM;
   - загружается конфигурация разрешённых номеров.
 
 - ожидается поступление запросов по протоколу из порта или SMS;
 
3. Протокол:
 - getname           - запрос  имени устройства
 - setname           - установка имени устройства
 - getvers           - запрос версии ПО устройства
 - getstat           - запрос статуса SIM карты: ready      - готов к выполнению полного спектра задач,
                                        no sim     - SIM не установлена,
									    no network - не возможна регистрация в сети;
									    no signal  - нет сигнала;
                                        no pin     - нет PIN кода.
 - getscn            - получение сценария работы;
 - setscn            - установка сценария работы;
 - getimei           - получить IMEI модема.
 - setpin            - установка PIN кода в конфигурацию.
 - getpin            - получение PIN кода из конфигурации.
 - enterpin          - ввести код сохранённый в конфигурации.
 - getreport                   - выдать разово отчёт.
 - reporton                 - включить выдачу отчётного сообщения.
 - reportoff                - выключить выдачу отчётного сообщения.
 - setreportint <0-86399> - установить интервал формирования отчётного сообщения в секундах.
   Формат отчётного сообщения:
   report <name> <imei> <scn> <status>
 - getreportint - получить интервал формирования отчётного сообщения
 - getgps - получить разово GPS
 - gpsreporton - включить отчёт GPS
 - gpsreportoff - выключить отчёт GPS
 - setgpsreportint <0-86399> - установить интервал формирования отчёта GPS в секундах
 - getgpsreportint  - получить интервал формирования отчёта GPS в секундах
 
4. Сценарии работы:
 - маяк, метка (Mark)          - в этом режиме устройство ожидает запросы SMS или с порта, формирует отчёты.
 - сложный маяк (Complex Mark) - в этом режиме устройство помимо базовых функций маяка общается с устройством подключённому к порту
